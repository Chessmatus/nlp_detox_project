This repository is a project that we had to do during NLP course of 10th semester at MIPT. 

## Code

`Evaluation` directory is copied from https://github.com/ducanhdt/ducanhhbtt-detoxification2025 repository. For better understanding of the results and inferences, you can refer to our report.
