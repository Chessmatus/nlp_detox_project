# NlpDetoxProject

This repository is a project that we had to do during NLP course of 10th semester at the MIPT. 

## Code

[Evaluation](https://github.com/Chessmatus/nlp_detox_project/tree/master/evaluation) directory is copied from https://github.com/ducanhdt/ducanhhbtt-detoxification2025 repository. In [notebook](https://github.com/Chessmatus/nlp_detox_project/blob/master/main.ipynb) all the experiments are conducted. For better understanding of the results and inferences, you can refer to our [report](https://github.com/Chessmatus/nlp_detox_project/blob/master/report/NLP_MIPT_Rubtsov_Eynullaev.pdf).

## Submissions 

All the submissions are available at [submissions](https://github.com/Chessmatus/nlp_detox_project/tree/master/submissions) directory. Moreover, via [link](https://www.codabench.org/competitions/15765/#/results-tab) you can see our best submission results. Team name is 'rubtsov_d'.
